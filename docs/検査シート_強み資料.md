# 検査シート作成アプリ ── 強み資料

## 概要
治療院の施術者が行う身体検査（立位・座位・上半身の左右差検査）をデジタル化し、段階的に原因を特定するアプリ。検査結果をもとにAIが診断結果を自動生成し、PDFレポートとして出力できる。患者ごとにカルテとして履歴管理が可能。

## URL
https://kensa-sheet-app.vercel.app

## 月額料金
5,500円（買い切り: 55,000円）

## ターゲットの悩み
1. 紙の検査シートだと保管・検索が大変で、過去データとの比較ができない
2. 姿勢分析アプリはカメラ撮影だけで、施術者の検査手技を記録できない
3. 患者への説明が口頭だけで、納得感を得にくい
4. ビフォーアフターの数値的な変化を見せられない
5. 検査結果から原因の特定に時間がかかる
6. カルテの電子化にコストをかけられない

## 競合3カテゴリ
1. AI姿勢分析アプリ（シセイカルテ、Sportip Pro等）: 月額1〜3万円。カメラ撮影型で施術者の検査手技は記録不可
2. 電子カルテシステム（RIPCLE、ほねぺん等）: 月額8,000〜15,000円。汎用カルテで検査特化の機能がない
3. 紙の検査シート: 0円。データ蓄積・比較・共有が不可能

## このアプリの立ち位置
「施術者の検査手技をデジタル化する」という新しいカテゴリ。カメラ撮影ではなく、施術者が実際に触れて確認した左右差・ランドマークの変位を段階的に記録し、AIが原因を特定する。

## 5つの強み
1. 段階的原因特定システム: 立位→座位→上半身の3段階検査で、乳様突起・肩甲下角・腸骨稜など主要ランドマークの左右差をワンタップで記録。AIが検査結果を統合して原因部位を自動特定
2. PDFレポート自動生成: 検査結果をプロフェッショナルなPDFレポートとして即時出力。患者への説明資料として活用可能。ビフォーアフターの変化を視覚的に提示
3. カルテ機能（検査履歴管理）: 患者ごとに検査履歴を自動保存。過去データとの比較で改善度を数値で把握。初診・再診の区分管理も対応
4. 痛みレベル（NRS）の定量記録: 0〜10のNRSスケールで主観的な痛みを客観データ化。経時的な痛みの変化をトラッキング
5. マルチテナント対応（SaaS型）: 院ごとにアカウント管理。Supabaseによるクラウドデータ保存。PWA対応でスマホ・タブレットからも操作可能
