カラダマップ プライバシーポリシー

プライバシーポリシー

最終更新日：2026年4月1日

大口神経整体院（以下「当院」）は、クラウド型検査記録サービス「カラダマップ」（以下「本サービス」）の提供にあたり、利用者および患者の個人情報の保護を重要な責務と認識し、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条（収集する個人情報）

当院は、本サービスの提供にあたり、以下の個人情報を収集することがあります。

1. 利用者（治療院）に関する情報

  • 院名、代表者氏名
  • 所在地、電話番号、メールアドレス
  • クレジットカード情報（決済代行会社を通じて処理）
  • ログイン情報（メールアドレス、パスワードのハッシュ値）
  • サービスの利用履歴、アクセスログ

2. 患者に関する情報（利用者が入力するデータ）

  • 氏名、年齢、性別
  • 検査記録（身体の状態、症状、検査結果等）
  • 施術履歴、来院記録
  • その他、利用者が本サービスに入力する情報

第2条（個人情報の利用目的）

当院は、収集した個人情報を以下の目的で利用します。

  1. 本サービスの提供、運営、維持および改善
  2. 利用者の本人確認およびアカウント管理
  3. 利用料金の請求および決済処理
  4. サービスに関するお知らせ、サポート対応
  5. 利用状況の分析によるサービス品質の向上（統計データとして匿名化のうえ利用）
  6. 規約違反の調査および対応
  7. 法令に基づく対応

第3条（患者データの取り扱い）

  1. 利用者が本サービスに入力する患者データは、利用者が管理責任を負います。
  2. 当院は、患者データに対して技術的に適切なセキュリティ対策（通信の暗号化、アクセス制御等）を講じます。
  3. 当院は、システムの保守運用に必要な場合を除き、患者データの内容にアクセスしません。
  4. 利用者は、患者データの入力にあたり、患者本人から個人情報の取り扱いについて適切な同意を得る責任を負います。

第4条（第三者提供）

当院は、以下の場合を除き、個人情報を第三者に提供しません。

  1. 本人の同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難な場合
  4. サービスの提供に必要な範囲で業務委託先（サーバー運営、決済代行等）に提供する場合。この場合、委託先に対して適切な監督を行います。

第5条（安全管理措置）

当院は、個人情報の漏洩、滅失、毀損の防止のため、以下の安全管理措置を講じます。

  • SSL/TLSによる通信の暗号化
  • データベースへのアクセス制御
  • パスワードのハッシュ化保存
  • 定期的なセキュリティ監査
  • 従業者への個人情報保護に関する教育

第6条（Cookieおよびアクセス解析）

  1. 本サービスでは、利用者の利便性向上およびサービス改善のため、Cookieを使用することがあります。
  2. Cookieにより個人を特定する情報は収集しません。
  3. 利用者は、ブラウザの設定によりCookieの受け入れを拒否することができます。ただし、一部の機能が正常に動作しなくなる場合があります。
  4. 当院は、アクセス解析ツールを使用してサービスの利用状況を把握することがあります。収集されるデータは統計情報として処理され、個人を特定することはありません。

第7条（個人情報の開示・訂正・削除）

  1. 利用者は、当院に対して自己の個人情報の開示、訂正、追加、削除、利用停止を請求することができます。
  2. 前項の請求があった場合、当院は本人確認を行ったうえで、合理的な期間内に対応します。
  3. 利用者が入力した患者データについては、利用者自身がサービス上で管理・削除できます。

第8条（データの保存期間）

  1. 利用者のアカウント情報およびデータは、サービス利用期間中保存されます。
  2. サービス解約後、当院は合理的な期間（原則30日間）経過後にデータを削除します。
  3. 法令により保存が義務付けられている情報については、所定の期間保存します。

第9条（本ポリシーの変更）

  1. 当院は、法令の改正やサービス内容の変更に伴い、本ポリシーを変更することがあります。
  2. 重要な変更を行う場合は、本サービス上での掲示またはメール等の方法により利用者に通知します。
