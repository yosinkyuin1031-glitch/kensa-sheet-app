カラダマップ プライバシーポリシー

プライバシーポリシー

制定日：2026年4月1日 / 最終改定日：2026年4月22日

大口神経整体院（以下「当院」）は、クラウド型検査記録サービス「カラダマップ」（以下「本サービス」）の提供にあたり、利用者および患者の個人情報（個人情報保護法上の要配慮個人情報を含む）の保護を重要な責務と認識し、個人情報の保護に関する法律その他関連法令を遵守し、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条（収集する個人情報）

当院は、本サービスの提供にあたり、以下の個人情報を収集することがあります。

1. 利用者（治療院）に関する情報

  • 院名、代表者氏名
  • 所在地、電話番号、メールアドレス
  • クレジットカード情報（決済代行会社を通じて処理）
  • ログイン情報（メールアドレス、パスワードのハッシュ値）
  • サービスの利用履歴、アクセスログ

2. 患者に関する情報（利用者が入力するデータ）

  • 氏名、年齢、性別
  • 検査記録（身体の状態、症状、検査結果等。個人情報保護法上の要配慮個人情報に該当）
  • 施術履歴、来院記録
  • その他、利用者が本サービスに入力する情報

※患者データは、利用者が個人情報取扱事業者として適法に取得し、本サービスに入力するものです。当院は、利用者の委託を受けてデータを処理する「委託先」の立場となります。要配慮個人情報については、利用者が患者本人から明示的な同意を取得のうえ登録するものとします。

第2条（個人情報の利用目的）

当院は、収集した個人情報を以下の目的で利用します。

  1. 本サービスの提供、運営、維持および改善
  2. 利用者の本人確認およびアカウント管理
  3. 利用料金の請求および決済処理
  4. サービスに関するお知らせ、サポート対応
  5. 利用状況の分析によるサービス品質の向上（統計データとして匿名化のうえ利用）
  6. 規約違反の調査および対応
  7. 法令に基づく対応

第3条（患者データの取り扱いにおける役割）

  1. 患者データについて、利用者が個人情報保護法上の「個人情報取扱事業者」としての地位に立ち、当院は利用者から取扱いの委託を受ける「委託先」としてこれを取り扱います。
  2. 利用者は、患者から個人情報の取得・利用・当院への委託について必要な同意を得たうえで、患者データを本サービスに登録するものとします。
  3. 要配慮個人情報（検査記録・既往歴等）の登録については、患者本人から明示的な同意を取得することが必要です。
  4. 当院は、システムの保守運用に必要な場合を除き、患者データの内容にアクセスしません。
  5. 当院は、委託された患者データを本サービスの提供目的以外に利用しません。

第4条（第三者提供・業務委託先）

当院は、以下の場合を除き、個人情報を第三者に提供しません。

  1. 本人の同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難な場合

サービス提供にあたり、以下の業務委託先を利用しています。いずれも当院が安全管理措置を確認のうえ、必要最小限の範囲で情報を委託しています。

  • Supabase Inc.（米国） - データベース・認証基盤。データは東京リージョン（日本国内）に保管。
  • Stripe, Inc.（米国） - 決済処理。PCI DSS Level 1準拠。当院はカード情報を保持しません。
  • Vercel Inc.（米国） - アプリケーションホスティング。
  • Resend（米国） - トランザクションメール送信。

第5条（安全管理措置）

当院は、個人情報の漏洩、滅失、毀損を防止するため、以下の安全管理措置を講じます。

技術的安全管理措置

  • SSL/TLSによる通信経路の暗号化
  • Supabase Row Level Security（RLS）による院ごとのデータ分離
  • パスワードのbcryptハッシュ化
  • 役割ベースアクセス制御（RBAC）
  • ログイン失敗回数制限（ブルートフォース対策）
  • アクセスログの記録・監視

物理的・組織的安全管理措置

  • データは東京リージョン（日本国内）のデータセンターに保管
  • Supabaseによる自動バックアップ
  • 管理者・委託先へのアクセス権限の必要最小限化
  • 定期的なセキュリティレビュー

人的安全管理措置

  • 従業者に対する個人情報取扱いの教育
  • 守秘義務の徹底

第5条の2（漏洩等発生時の対応）

個人情報の漏洩、滅失または毀損が発生し、かつ個人の権利利益を害するおそれが大きいと認められる場合、当院は個人情報保護委員会への報告および影響を受ける本人・利用者への通知を、事故認知後速やかに（原則72時間以内を目安に）行います。

第6条（Cookieおよびアクセス解析）

  1. 本サービスでは、利用者の利便性向上およびサービス改善のため、Cookieを使用することがあります。
  2. Cookieにより個人を特定する情報は収集しません。
  3. 利用者は、ブラウザの設定によりCookieの受け入れを拒否することができます。ただし、一部の機能が正常に動作しなくなる場合があります。
  4. 当院は、アクセス解析ツールを使用してサービスの利用状況を把握することがあります。収集されるデータは統計情報として処理され、個人を特定することはありません。

第7条（個人情報の開示・訂正・削除）

  1. 利用者は、当院に対して自己の個人情報の開示、訂正、追加、削除、利用停止を請求することができます。
  2. 前項の請求があった場合、当院は本人確認を行ったうえで、合理的な期間内に対応します。
  3. 利用者が入力した患者データについては、利用者自身がサービス上で管理・削除できます。

第8条（データの保存期間）

  1. 利用者のアカウント情報およびデータは、サービス利用期間中保存されます。
  2. サービス解約後、当院は合理的な期間（原則30日間）経過後にデータを削除します。
  3. 法令により保存が義務付けられている情報については、所定の期間保存します。

第9条（本ポリシーの変更）

  1. 当院は、法令の改正やサービス内容の変更に伴い、本ポリシーを変更することがあります。
  2. 重要な変更を行う場合は、本サービス上での掲示またはメール等の方法により利用者に通知します。
