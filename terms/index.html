カラダマップ 利用規約

利用規約

制定日：2026年4月1日 / 最終改定日：2026年4月22日

この利用規約（以下「本規約」）は、大口神経整体院（以下「当院」）が提供するクラウド型検査記録サービス「カラダマップ」（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただくすべての利用者（以下「利用者」）は、本規約および別途定めるプライバシーポリシーに同意のうえ、本サービスを利用するものとします。

第1条（定義）

  1. 「本サービス」とは、当院が提供するクラウド型の検査シート記録・管理アプリケーション「カラダマップ」およびこれに付随する一切の機能を指します。
  2. 「利用者」とは、本規約に同意のうえ本サービスの利用登録を行った治療院、整骨院、整体院等の事業者および個人を指します。
  3. 「患者データ」とは、利用者が本サービスに入力・保存する患者の検査記録、氏名、症状、健康情報等、個人情報保護法上の要配慮個人情報を含む情報を指します。

第2条（利用登録）

  1. 本サービスの利用を希望する方は、当院所定の方法により利用登録を申請していただきます。
  2. 当院は、以下の場合に利用登録を拒否することがあります。
    • 登録内容に虚偽の事項が含まれる場合
    • 過去に本規約に違反したことがある場合
    • その他、当院が利用登録を適当でないと判断した場合

第3条（アカウントの管理）

  1. 利用者は、自己の責任においてアカウント情報（メールアドレス、パスワード等）を適切に管理するものとします。
  2. 利用者は、アカウント情報を第三者に譲渡、貸与、共有してはなりません。
  3. アカウント情報の管理不十分、第三者の使用等による損害について、当院は一切の責任を負いません。
  4. 利用者は、不正アクセスまたはその兆候を認知した場合、速やかに当院に通知するものとします。

第4条（料金および支払い）

  1. 本サービスの利用料金は、当院が別途定める料金表（税込）に基づき、決済代行事業者 Stripe を通じて毎月自動課金されます。
  2. 月額プランの課金日は契約開始日と同日とし、以降毎月同日に自動更新されます。
  3. 支払いの遅延が発生した場合、当院は催告後にサービスの提供を一時停止できるものとします。
  4. 支払済みの利用料金は、法令に別段の定めがある場合を除き、返金いたしません。
  5. 当院は、30日前までに通知のうえ、料金を変更することがあります。通知後も利用を継続した場合は新料金に同意したものとみなします。

第5条（最低契約期間・解約・早期解約金）

  1. 月額プランの最低契約期間は、契約開始月を1ヶ月目として6ヶ月とします。
  2. 最低契約期間満了後は、利用者が解約の意思表示を行うまで自動的に1ヶ月単位で更新されます。
  3. 利用者は、次回請求日の7日前までに管理画面または書面により解約の意思を当院に通知することで、本サービスを解約できます。
  4. 最低契約期間内に利用者都合で解約する場合、残存期間分の月額料金を早期解約金として一括請求します（Stripe請求書にて発行、発行日から14日以内にお支払いください）。
  5. 月途中の解約であっても日割り返金はいたしません。
  6. 一度解約したメールアドレス・事業者での再登録は、当院が認めた場合を除き原則として受け付けません。

第6条（禁止事項）

利用者は、本サービスの利用にあたり、以下の行為を行ってはなりません。

  1. 法令または公序良俗に反する行為
  2. 犯罪行為に関連する行為
  3. 本サービスのサーバーまたはネットワークに過度な負荷をかける行為、スクレイピング、クローリング
  4. 本サービスの運営を妨害する行為、不正アクセス、コンピュータウイルスの送信
  5. 他の利用者の情報を不正に収集・蓄積する行為
  6. 第三者になりすます行為、アカウントの譲渡・貸与・共有
  7. 本サービスを本来の目的以外に使用する行為
  8. 本サービスのリバースエンジニアリング、逆コンパイル、逆アセンブル等の行為
  9. 反社会的勢力に該当する行為、またはこれを利用する行為
  10. 当院の知的財産権を侵害する行為
  11. その他、当院が不適切と判断する行為

第7条（反社会的勢力の排除）

利用者は、自己および自己の役員・従業員等が、暴力団、暴力団関係企業、総会屋、社会運動標榜ゴロ、特殊知能暴力集団その他の反社会的勢力に該当しないこと、および反社会的勢力と一切関係を有しないことを表明し、保証します。利用者がこれに違反した場合、当院は何らの催告なく直ちに本契約を解除できるものとします。

第8条（サービスの提供・変更・停止）

  1. 当院は、利用者に事前に通知することなく、本サービスの内容を変更し、または本サービスの提供を中止もしくは停止することがあります。
  2. 以下の事由に該当する場合、当院は本サービスの全部または一部を停止することがあります。
    • システムの保守点検または更新を行う場合
    • 地震、火災、停電、天災等の不可抗力により提供が困難な場合
    • 外部サービス（Supabase、Stripe、Vercel等）の障害による場合
    • その他、当院がサービスの停止を必要と判断した場合
  3. 本サービスの重大な変更・終了については、原則として30日前までに通知いたします。
  4. 当院は、本サービスの提供停止または中止により利用者に生じた損害について、一切の責任を負いません。

第9条（知的財産権）

  1. 本サービスに関する著作権、特許権、商標権その他一切の知的財産権は、当院または正当な権利者に帰属します。
  2. 利用者は、本サービスを通じて提供されるコンテンツを、当院の事前の書面による承諾なく、複製、転載、改変、販売等してはなりません。
  3. 利用者が本サービスに入力した患者データの権利は、利用者または当該患者に帰属します。ただし、当院はサービス提供および改善に必要な範囲で、匿名化・統計化したデータを利用できるものとします。

第10条（患者データの取り扱いに関する特約）

  1. 患者データについて、利用者は個人情報保護法上の「個人情報取扱事業者」の地位に立ち、当院は利用者から委託を受けてこれを取り扱う「委託先」の地位に立つものとします。
  2. 利用者は、患者データを本サービスに入力するにあたり、患者から個人情報の取得・利用・当院への委託について必要な同意を取得する責任を負います。
  3. 検査記録・既往歴・健康情報等、個人情報保護法上の要配慮個人情報を本サービスに登録するにあたっては、利用者は患者本人からの明示的な同意を取得したうえで行うものとします。
  4. 当院は、委託された患者データを本サービスの提供目的以外に利用せず、次条に定める安全管理措置を講じて取り扱います。
  5. 患者からの開示・訂正・削除・利用停止等の請求は、まず利用者が一次対応を行うものとし、必要に応じて当院が技術的に協力します。

第11条（安全管理措置）

当院は、患者データを含む個人情報の漏洩、滅失、改ざんを防止するため、以下の安全管理措置を講じています。

  1. SSL/TLSによる通信経路の暗号化
  2. Supabase Row Level Security による院ごとのデータアクセス制限
  3. パスワードのbcryptハッシュ化保存
  4. データベースの国内リージョン（東京）での保管
  5. Supabaseによる自動バックアップ
  6. 役割ベースアクセス制御（RBAC）およびアクセスログの記録
  7. 決済カード情報は当院では保持せず、Stripeに直接送信（PCI DSS Level 1準拠）
  8. ログイン失敗回数制限（ブルートフォース対策）
  9. 業務委託先の選定および定期的なセキュリティレビュー

第12条（漏洩等発生時の対応）

当院は、個人情報の漏洩、滅失または毀損が発生した場合、速やかに調査するとともに、個人情報保護委員会への報告および影響を受ける利用者への通知を、事故認知後速やかに（原則72時間以内を目安に）行います。

第13条（免責・損害賠償の上限）

  1. 当院は、本サービスに事実上または法律上の瑕疵（安全性、信頼性、正確性、完全性、有効性、特定目的への適合性、権利侵害等に関する欠陥を含みますが、これらに限りません）がないことを明示的にも黙示的にも保証しておりません。
  2. 当院の責に帰すべき事由により利用者に損害が発生した場合、当院が利用者に対して負う損害賠償責任は、当院に故意または重大な過失がある場合を除き、損害発生時点から遡って直近3ヶ月間に利用者が当院に現実に支払った利用料金の総額を上限とし、逸失利益、間接損害、特別損害、懲罰的損害については責任を負いません。
  3. 本サービスに基づく検査記録は、あくまで参考情報であり、医療行為・診断行為ではなく、医学的判断・施術判断の最終責任は利用者に帰属します。
  4. 利用者は、自己の責任において定期的にデータをエクスポート・バックアップするものとし、当院はデータの完全性・利用可能性を保証するものではありません。

第14条（利用制限および登録抹消）

  1. 当院は、利用者が以下のいずれかに該当する場合、事前の通知なく利用制限または登録抹消を行うことがあります。
    • 本規約のいずれかの条項に違反した場合
    • 料金の支払いを怠った場合
    • 反社会的勢力に該当する、またはこれと関係を有することが判明した場合
    • その他、当院が利用を適当でないと判断した場合
  2. 前項により契約が解除された場合でも、早期解約金（第5条）の支払義務は免除されません。

第15条（規約の変更）

  1. 当院は、民法第548条の4の規定に基づき、必要と判断した場合、利用者に通知のうえ本規約を変更することがあります。
  2. 変更後の規約は、当院が別途定める場合を除き、本サービス上に表示した時点で効力を生じるものとします。
  3. 規約変更後も本サービスの利用を継続した場合、利用者は変更後の規約に同意したものとみなされます。

第16条（準拠法・管轄裁判所）

  1. 本規約の解釈にあたっては、日本法を準拠法とします。
  2. 本サービスに関して紛争が生じた場合、横浜地方裁判所を第一審の専属的合意管轄裁判所とします。
